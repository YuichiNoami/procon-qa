# procon-qa

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

エラーが出た場合、以下のコマンドを実行する
```
rm -rf node_modules
npm install -g npm@latest
npm i core-util-is
```

### Compiles and minifies for production
```
npm run build
```
/dist 以下に *html, *js, *css ファイルが生成される

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
commit する前は必ず実行する
